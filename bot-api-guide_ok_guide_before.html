BOT / API GUIDE

텔레그램 BOT TOKEN / CHAT ID / 거래소 API 만들기

반자동·자동 신청 전 필요한 정보를 초보자 기준으로 순서대로 안내합니다.

텔레그램부터 보기거래소 API 보기등록 페이지로 이동

먼저 꼭 확인하세요

출금 권한은 절대 켜지 마세요. K-EDGE 연동에는 거래 권한만 사용합니다.

반자동과 자동 시스템은 유저 본인의 텔레그램 개인봇과 거래소 API를 연결해서 작동합니다. 아래 순서대로 만들고, 등록 페이지에 붙여넣으면 됩니다.

STEP 1. 텔레그램 BOT TOKEN 만들기

  1. 텔레그램 앱을 엽니다.
  2. 검색창에 BotFather를 검색합니다.
  3. 파란 체크가 있는 BotFather를 선택합니다.
  4. 채팅창에 /start 입력 후 전송합니다.
  5. 채팅창에 /newbot 입력 후 전송합니다.
  6. 봇 이름을 입력합니다. 예: KedgeSemiBot
  7. 봇 아이디를 입력합니다. 반드시 bot으로 끝나야 합니다. 예: kedge_semi_bot
  8. 마지막에 나오는 긴 문자가 BOT TOKEN입니다. 전체 복사하세요.
BOT TOKEN 예시: 1234567890:AAxxxxxxxxxxxxxxxxxxxxxxxx
Telegram · BotFather 예시 화면
BotFather를 시작하려면 /start 를 입력하세요.
/newbot
새 봇 이름을 입력하세요.
KedgeSemiBot
봇 아이디를 입력하세요. 반드시 bot으로 끝나야 합니다.
kedge_semi_bot
완료! 아래 토큰을 복사하세요.
1234567890:AAxxxxxx

STEP 2. CHAT ID 찾기

  1. 방금 만든 봇을 텔레그램에서 검색합니다.
  2. 봇 채팅방에 들어가서 START 버튼을 누릅니다.
  3. 인터넷 주소창에 아래 형식으로 입력합니다.
https://api.telegram.org/bot여기에_BOT_TOKEN_붙여넣기/getUpdates
  1. 열린 화면에서 "chat" 근처의 "id" 숫자를 찾습니다.
  2. 그 숫자가 CHAT ID입니다.
CHAT ID 예시: 123456789 또는 -1001234567890
getUpdates 예시 화면
{ "message": { "chat": {
  "id": 123456789,
  "first_name": "KEDGE"
} } }

파란색 숫자 부분을 CHAT ID 칸에 입력합니다.

봇 채팅방에서 START를 누르지 않으면 CHAT ID가 안 나올 수 있습니다.

STEP 3. 국내 거래소 API 만들기

국내 거래소 API는 현물 주문 확인/진입에 사용됩니다. 출금 권한은 절대 사용하지 않습니다.

업비트거래 권한 ON출금 권한 OFF
빗썸거래 권한 ON출금 권한 OFF
코인원거래 권한 ON출금 권한 OFF

등록 페이지에 입력할 값

  • API KEY 또는 ACCESS KEY
  • SECRET KEY

STEP 4. 해외 거래소 API 만들기

해외 거래소 API는 선물 헷지 포지션 확인/진입에 사용됩니다. 출금 권한은 절대 사용하지 않습니다.

MEXCSpot/Futures Trade ONWithdraw OFF
Gate.ioTrade ONWithdraw OFF
BitgetTrade ONWithdraw OFF
BingXTrade ONWithdraw OFF

반자동 / 자동 신청 전 준비물

  • 텔레그램 BOT TOKEN
  • 텔레그램 CHAT ID
  • 국내 거래소 API KEY / SECRET KEY
  • 해외 거래소 API KEY / SECRET KEY
API 생성 시 출금 권한이 켜져 있으면 등록하지 마세요. 반드시 출금 OFF 상태로 다시 만들어주세요.
