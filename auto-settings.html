✅ K-EDGE AUTO 승인완료

알람은 기본 ON, 자동매매는 사용자가 직접 ON으로 전환해야 시작됩니다.

승인정보 확인중...

1. 운용 방식

고정금액은 1회 진입금액을 직접 유지하고, 총자산 복리는 빗썸 잔고조회 금액을 분할 수로 나눠 계산합니다.

아직 조회하지 않았습니다.

2. 해외거래소 운용 선택

API는 삭제하지 않고 보존합니다. OFF로 둔 거래소는 자동진입 대상에서만 제외됩니다.

※ API가 등록되지 않은 거래소는 선택할 수 없습니다. API 수정/추가는 등록 페이지에서 진행하세요.

3. 자동매매

ON으로 저장하면 조건 충족 시 자동진입을 시도합니다. 빗썸 최소주문금액 5,000원 미만이면 저장하지 마세요.
