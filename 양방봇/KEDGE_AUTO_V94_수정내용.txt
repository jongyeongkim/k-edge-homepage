K-EDGE AUTO V9.4 수정내용

1. 반자동/금액선택/진입실행 버튼 폐기
- AMT / RESET / MAX / RUN / CONFIRM / CANCEL callback은 동작하지 않도록 차단
- AUTO 전용: 정지/재시작 버튼만 유지

2. auto_settings 연동 추가
- SUPABASE_AUTO_SETTINGS_TABLE 기본값: auto_settings
- user_id/member_id/request_id/id/uid/telegram_id/tg_chat_id/chat_id 기준으로 승인회원과 설정 매칭
- service_enabled=false면 스킵
- auto_enabled=false면 자동진입 없이 알림만 전송
- auto_enabled=true면 자동진입 진행

3. 유저별 금액 계산 추가
- capital_krw / split_count 값을 읽어 진입금액 계산
- 값이 없으면 기존 PAPER_DOMESTIC_BASE_KRW / DOMESTIC_COMPOUND_DIVISOR fallback

4. 정지 버튼 안전 수정
- 기존: 신규진입 정지 + 전체 포지션 종료
- 변경: 신규진입만 정지, OPEN 포지션 감시/익절/손절은 유지

5. OPEN 복구 유지
- recover_open_positions_from_paper_entries() 유지

6. 실행 BAT 추가
- start_kedge_auto_v94.bat
- storage_worker + BINGX/BITGET/GATE/MEXC 4파일 실행
- MEXC만 ENABLE_CALLBACK_POLLER=true

7. 거래소명 출력 수정
- 각 파일 시작문구가 자기 라우트명으로 표시되도록 수정
