BOT / API GUIDE

반자동 · 자동 연동 가이드

VIP Lite / VIP Pro 신청 전 필요한 텔레그램 BOT TOKEN, CHAT ID, 국내/해외 거래소 API KEY 생성 및 확인 방법입니다.

보안 주의

국내/해외 API KEY 모두 출금 권한은 절대 켜지 마세요. 읽기/거래 권한만 사용하고, IP 제한이 가능하면 반드시 설정하세요.

1

텔레그램 BOT TOKEN 만들기

  1. 텔레그램에서 @BotFather 검색
  2. /newbot 입력
  3. 봇 이름 입력
  4. 봇 아이디 입력
    예: kedge_my_bot
  5. BotFather가 보내주는 BOT TOKEN 복사
예시

8982xxxx:AAxxxxxx

2

CHAT ID 확인하기

  1. 방금 만든 봇에게 아무 메시지 전송
  2. 아래 주소 형식으로 접속
  3. chat → id 값 확인
https://api.telegram.org/bot내BOT_TOKEN/getUpdates

개인 채팅 ID는 숫자 형태, 그룹/채널 ID는 보통 -100으로 시작합니다.

3

국내 거래소 API 만들기

  1. 업비트 / 빗썸 / 코인원 로그인
  2. API 관리 메뉴 이동
  3. 새 API 생성
  4. 읽기 / 거래 권한만 체크
  5. 출금 권한은 반드시 OFF
  6. API KEY 또는 Access Key / Secret Key 복사
4

해외 거래소 API 만들기

  1. MEXC / Bitget / BingX / Bybit 등 로그인
  2. API Management 이동
  3. 새 API 생성
  4. Futures/Contract 거래 권한 확인
  5. 출금 권한은 반드시 OFF
  6. API KEY / SECRET KEY 복사
!

API 권한 체크 기준

  • 읽기 권한: ON
  • 거래 권한: ON
  • 선물/계약 거래: 해외 선물 사용 시 ON
  • 출금 권한: OFF
  • IP 제한: 가능하면 ON
5

신청 전 최종 확인

  1. BOT TOKEN 정상 여부
  2. CHAT ID 정상 여부
  3. 국내 API KEY / SECRET 정상 여부
  4. 해외 API KEY / SECRET 정상 여부
  5. 거래소 선택값 확인
