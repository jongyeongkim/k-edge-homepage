반자동·VIP 알림 사용 가이드

텔레그램 알림을 처음 보는 사용자도 이해할 수 있도록, 실제 VIP 알림 화면 기준으로 용어와 안전장치를 정리했습니다.

🤖 @Kedge0203bot /start 필수 🛡 진입 직전 재검사 🧱 0.5% 실체결벽 기준 ⚖️ 현물 매수 + 선물 숏

🤖 반자동 사용 전 필수

  • 텔레그램에서 @Kedge0203bot 검색
  • 봇 채팅방에서 /start 입력
  • 홈페이지에서 반자동 신청
  • 관리자 승인 후 개인 알림과 버튼 사용
🤖 Kedge0203bot 연결 등록 신청

⚠️ /start가 필요한 이유

텔레그램 봇은 사용자가 먼저 대화를 시작해야 개인 DM을 보낼 수 있습니다. /start를 하지 않으면 개인 알림, 반자동 버튼, 진입/청산 알림이 작동하지 않습니다.

/start 미입력 시 개인 알림 / 반자동 / 버튼 기능이 작동하지 않습니다.

📱 실제 VIP 텔레그램 알림 읽는 법

아래 화면은 실제 VIP방에 전송되는 현물·선물 양방 후보 알림 예시입니다.

알림 상단: 코인, 전략, 현물/선물 거래소와 기준가
중간 지표: 괴리율, BTC 기준 프리미엄, 예상 수익구간, 벽
하단 지표: 실체결 가능금액, 권장 레버리지, 손절 기준
① BTC 기준 프리미엄

현재 시장 전체의 프리미엄 기준입니다. BTC가 이미 +1% 비싸다면, 다른 코인의 +3% 괴리는 실제로 +2% 정도만 의미가 있습니다.

실제엣지 = 코인 괴리율 - BTC 기준 프리미엄
② 코인 괴리율

해당 코인의 국내 현물 가격과 해외 선물 가격 차이입니다. 예: 현물보다 선물이 높으면 현물 매수 + 선물 숏 후보가 됩니다.

③ 예상 수익구간

수수료, 슬리피지, 시장 변동을 감안한 예상 구간입니다. 확정 수익이 아니라 실시간으로 변하는 참고 구간입니다.

④ 현물벽 / 선물벽 0.5%

현재 가격 기준 0.5% 범위 안에 쌓여 있는 매수·매도 물량입니다. 즉, 실제로 체결될 가능성이 있는 금액을 의미합니다.

⑤ 최종 진입 가능금액

현물벽, 선물벽, 거래소 최대 진입금액 중 가장 작은 값으로 계산합니다. 많이 보이는 기회라도 실제 체결 가능금액이 작으면 작게 표시됩니다.

최종 진입 가능 = MIN(현물벽, 선물벽, 거래소 최대)
⑥ 펀딩비

선물 숏 포지션 유지 비용 또는 수령 금액입니다. 음수(-)는 시간마다 비용이 차감될 수 있고, 양수(+)는 수령 가능성이 있습니다.

🛡 K-EDGE 진입 안전장치

반자동/자동은 신호 발생 즉시 무조건 진입하지 않습니다. 사용자가 진입을 누르는 순간 다시 검사합니다.

📊

실제엣지 2% 이상

알림 당시 조건이 좋아도 클릭 순간 실제엣지가 2% 미만이면 진입하지 않습니다.

🧱

슬리피지 0.5% 이하

진입 직전 현물벽과 선물벽을 다시 확인해 0.5% 범위 안에서 체결 가능한지 검사합니다.

👥

동시 진입 방어

여러 명이 동시에 눌러도 먼저 체결된 금액을 제외하고 남은 금액 기준으로 판단합니다.

💰

잔고 확인

국내 KRW 잔고와 해외 USDT 잔고가 부족하면 진입하지 않고 사유를 표시합니다.

🔑

API 등록 확인

국내/해외 API가 없거나 잘못 등록된 경우 진입하지 않습니다.

🚫

중복 진입 제한

이미 해당 코인 포지션이 있는 유저에게는 신규 진입 알림 대신 청산/경고 알림만 전송됩니다.

🤖 반자동 사용 흐름

1

개인 DM 수신

승인회원에게만 Kedge0203bot 개인 DM으로 후보가 전송됩니다.

2

금액 버튼 선택

+1만, +5만, +10만, +50만, +100만, +500만 버튼으로 빠르게 금액을 누적합니다.

3

진입 실행

진입 버튼을 누르면 실제엣지, 슬리피지, 잔고, 남은금액을 재검사합니다.

4

청산 감시

진입 성공 후 자동익절, 경고, 손절감시가 작동합니다.

✅ 실패 사유는 이렇게 표시됩니다

상황표시되는 안내의미
실제엣지 부족❌ 현재 실제엣지 부족알림 후 시간이 지나 수익 가능 구간이 줄어든 상태
슬리피지 초과❌ 현재 0.5% 벽 부족실제 체결 가능한 물량이 줄어든 상태
남은금액 부족❌ 남은 진입 가능금액 부족다른 유저가 먼저 진입해 남은 금액이 부족한 상태
API 없음❌ 국내/해외 API 미등록거래소 API가 등록되지 않아 진입 불가
잔고 부족❌ 국내 KRW 또는 해외 USDT 부족선택금액 대비 잔고가 부족한 상태

📉 자동청산 / 손절감시 기준

익절

실제엣지가 +0.5% ~ +0.3% 구간으로 회귀하면 유연하게 청산을 시도합니다. 익절은 정확한 한 지점보다 체결 우선으로 관리합니다.

1차 경고 / 2차 강경고

진입엣지 대비 +4% 악화 시 1차 경고, +6% 악화 시 강경고를 전송합니다.

자동손절

진입엣지 대비 +8% 이상 악화가 15분 동안 계속 유지되면 자동손절 기준으로 판단합니다. 중간에 +8% 아래로 회귀하면 손절감시는 해제됩니다.

🔐 API 보안 안내

  • 거래소 API는 출금 권한 OFF 상태로 등록하세요.
  • 가능하면 거래 권한만 허용하고, 출금/이체 권한은 사용하지 마세요.
  • 거래소에서 제공하는 경우 IP 제한을 설정하는 것을 권장합니다.
  • K-EDGE는 진입 전 조건 재검사 후 조건 미충족 시 진입하지 않도록 설계되어 있습니다.
